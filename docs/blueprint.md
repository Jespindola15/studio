# **App Name**: Mediup Ascent

## Core Features:

- Homepage Sections Display: Render and display all static content sections on the homepage including Hero, Value Proposition, About Us, and Call to Action, designed to be modern and engaging.
- Interactive Services Showcase: Present detailed information for each service (Landing Pages, Corporate Web Design, Redesign, Social Media Visuals) in modern, interactive cards with icons and descriptions.
- Dynamic Portfolio Gallery: Implement a modern grid layout displaying visual examples of web projects with preview images, project names, and categories.
- Simple Process Timeline: Visualize the agency's work process through a minimal timeline, explaining steps from idea to publication.
- Quote Request Form: Provide a clean, user-friendly form for visitors to submit inquiries and request a personalized budget/quote, integrated with a backend service for submissions.
- Smooth Navigation and Responsiveness: Ensure seamless navigation across sections with smooth scrolling and adapt the layout and elements responsively for optimal viewing on all devices.

## Style Guidelines:

- Primary Color: Electric blue (#3B82F6) representing innovation and technology. (HSL: 217, 91%, 60%)
- Background Color: A very dark, subtly blue-tinted grey (#10151A) for a futuristic and premium feel. (HSL: 217, 20%, 8%)
- Accent Color: Bright cyan (#22D3EE) for vibrant highlights and calls to action. (HSL: 187, 89%, 53%)
- Supporting Colors: Utilize the user-suggested palette of deep black (#0B0B0F), dark grey (#111217), medium grey (#1A1C23), and soft white (#F5F7FA) for text and subtle contrasts.
- Gradients: Apply subtle, modern gradients combining blue, violet, and cyan hues to evoke a futuristic aesthetic on backgrounds and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its clean, technological, and professional appearance, suitable for all textual content.
- Minimalist Icons: Utilize clean, modern, and conceptual icons to accompany service descriptions and value propositions, enhancing the professional and digital aesthetic.
- Clean and Spacious Layout: Embrace ample white space, distinct section separation, and generous padding to maintain a minimalist, elegant, and SaaS-inspired interface.
- Modern Element Styling: Incorporate rounded borders, subtle soft shadows, and light glassmorphism effects for a premium and futuristic look.
- Smooth Page Transitions and Microinteractions: Implement subtle load-in animations, professional hover effects, and smooth scroll-based animations to create a polished and dynamic user experience.
